萝莉有三好，清音体柔易推倒。

四十未婚的苦逼程序员顾延在出差途中飞机失事，醒过来以后变成了一只小萝莉。

萝莉名叫初春静香（ういはる　しずか），有一个双胞胎姐姐名叫初春饰利（ういはる　かざり）。现就读于学园都市栅川副小四年级。

然而作为妹妹却更像姐姐是肿么回事口牙！

在这有着“超自然”力量的举目无亲的异世界里，作为一个 Lv.1 的软萌的小萝莉，初春静香觉得自己还是学着姐姐一样卖萌算了。

只不过在这 230 万人齐聚的是非之地里，小萝莉不断地与黑色的人和事阴差阳错地相逢。

回过头来，此身已凭借非神之躯，上听天意。
